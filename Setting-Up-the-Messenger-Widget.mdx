## 🔄 Prerequisites

Before creating a Messenger Widget, make sure you’ve completed the following:

- ✅ Created a **Knowledge Base** (with your FAQs, documents, or resources)
- ✅ Built an **Assistant** (the AI agent that powers your chatbot)

> 📘 Need help?\
> Follow this [step-by-step guide](#) to learn how to create your knowledge base and Assistant.

---

## 🧩 Creating the Messenger Widget

Once your Assistant is ready, you can proceed to create and configure the Messenger widget.

### 🛠 Step-by-Step Instructions

1. Log in to your [Insighto.ai Dashboard](https://dashboard.insighto.ai)
2. Navigate to **AI Agents \> Widgets**
3. Click **Add a Widget**

### Fill in the Widget Details

In the widget creation form, fill out the following:

- **Provider**: Select `Messenger`
- **Widget Type**: Choose the appropriate display type: (e.g. Messenger)
- **Name**: Give your widget a meaningful name (e.g., “Website Messenger Bot”)
- **Description**: Add a short internal description (e.g., “FB chat for main website users”)

---

## 🔗 Connect to Facebook Messenger

To allow the widget to send and receive messages via your Facebook Page:

1. Under **Messenger Configuration**, select your configured **Facebook App** or **Messenger Settings**\
   (Make sure this is already set up within your Insighto.ai Facebook Integration)
2. Choose the **Facebook Page** that is connected to your business
3. Ensure the page is properly authorized and connected via Meta

> 💡 If you haven’t connected your Facebook account yet, you will need to go to `Settings > Channels > Messenger` and follow the authentication and permission steps.

---

## ✅ Final Step: Submit

After entering all the required fields:

- Click **Submit** to create your Messenger Widget
- You’ll now see the new widget listed under **AI Agents \> Widgets** with its associated details and status

---

## 🌐 Deploying the Messenger Widget

After setup, you can embed the Messenger Widget on your website or web app for users to interact with your bot.

### To embed:

- Copy the **embed code** provided after widget creation
- Paste it into the HTML of your website — typically in the `<head>` tag or using a tag manager

> 🧠 **Note**: For CMS platforms like WordPress, Wix, or Webflow, you can use a custom code block or integrate it using platform-specific plugins.

---

## 🧪 Testing and Going Live

Before launching:

- Test the Messenger widget on your site
- Send a few messages to check if the responses are accurate
- Make sure your Facebook Page is published and allows messaging

---

## ❓ Troubleshooting

Here are some common issues and solutions:

| Issue                   | Solution                                                                        |
| ----------------------- | ------------------------------------------------------------------------------- |
| Widget not showing      | Make sure the embed code is correctly placed and the Facebook Page is connected |
| Messages not delivering | Verify Messenger channel is properly set up and access tokens are valid         |
| No response from bot    | Ensure your Assistant is linked to the widget and trained with a knowledge base |
